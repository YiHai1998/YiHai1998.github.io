

网络把许多计算机连接起来。

互联网则是网络的网络，它是通过路由器将许多网络连接起来的。

与网络相连的计算机成为`主机`。





# 互联网的三个子系统

https://www.bilibili.com/video/BV1JV411t7ow?p=4

```markdown
1. 网络边缘 edge
		- 主机/应用程序
2. 接入网、物理媒体 access
		- 将边缘接入到网络核心
3. 网络核心 core
		- 定义：core是全球范围内的，几十亿个节点之间的，瞬时的切换开关。core不可能只有1个，而是由一堆小的路由器交换机、连接的链路锁组成的分布式系统
		- 作用：数据交换
```
